Timer

Help:

Tap the Timer(purple) button and type the number of minutes you want to start then click "OK"

The the play(green) button to start the timer

Tap the pause(pinkred) button to pause the timer

To start another timer, click the Timer(purple) button again and type the number you want

To reset all, just refresh/reload the page
